# PC UI Pages

The Note Assistant desktop UI follows the Pencil prototype with these pages:

| ID    | Page                        | Description                             |
|-------|-----------------------------|-----------------------------------------|
| PC-01 | Home / Note Overview        | Main view with sidebar, list, and panel |
| PC-02 | Create Note                 | Manual note creation form               |
| PC-03 | Edit Note                   | Manual note editing                     |
| PC-04 | Delete Confirmation         | Confirm delete dialog                   |
| PC-05 | Search Results              | Fuzzy search results display            |
| PC-06 | Assistant Idle              | Voice assistant standby panel           |
| PC-07 | Voice Create Note           | Voice-driven note creation              |
| PC-08 | Voice Search                | Voice-driven note search                |
| PC-09 | Voice Update Note           | Voice-driven note update                |
| PC-10 | Voice Delete / Multi-select | Voice-driven delete with confirmation   |
| PC-11 | Settings                    | Application settings and connection info |

*Detail page descriptions will be added in Phase 3 when the QML UI is built.*
