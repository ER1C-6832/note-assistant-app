# Phase 9.1 已知问题

## 真实语音链路

1. 首次点击说话时可能需要准备音频设备，耗时略长。
2. Windows Defender / 虚拟机冷缓存可能影响首次启动速度。
3. 麦克风权限或默认输入设备异常时，真实语音链路可能不可用。
4. 登录预热默认关闭。需要更快启动时，可在设置页开启“开机后后台准备”。

## Mock 模式

1. Mock 模式不调用真实 LLM / ASR / TTS。
2. Mock 模式只演示前端 UI、结果卡片、多候选确认、删除确认和失败提示。
3. Mock 模式需要真实 Notes API 才能展示便签列表和演示数据。
4. 运行 Mock 前需要确保真实 Sidecar 没占用 17890 / 17891 端口。
