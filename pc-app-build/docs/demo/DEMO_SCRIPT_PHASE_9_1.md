# Phase 9.1 Demo 演示脚本

## 主路径：真实语音链路

1. 运行：

```bat
scripts\start_all_integrated.bat
```

2. 检查设置页：
   - 便签服务已连接。
   - 本地助手已连接。
   - 语音助手已就绪。

3. 演示手动便签：
   - 新增一条便签。
   - 搜索“项目例会”。
   - 编辑标签。
   - 删除并在回收站恢复。

4. 演示语音按钮：
   - 点击右下角“点击说话”。
   - 说：“新增一条便签，明天十点项目例会。”
   - 展示结果卡片。

5. 演示多候选：
   - 说：“修改周五例会。”
   - 展示多候选确认 UI。
   - 选择其中一条。

6. 演示防误删：
   - 说：“删除临时删除测试便签。”
   - 展示删除二次确认。
   - 确认后删除。

## 备用路径：Mock 语音桥接

如果真实语音链路临场不稳定，运行：

```bat
scripts\start_demo_mock_mode.bat
```

然后直接点击右下角语音按钮。Mock 会循环演示：

1. 语音执行结果卡片。
2. 多候选确认。
3. 删除二次确认。
4. 失败提示。
